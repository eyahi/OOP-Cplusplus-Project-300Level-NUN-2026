Personal Diary App
Description: This is a desktop application made in c++ that allows the user to create, edit and save journal entries, it tracks the dates each entry was created and secures it as well

Compilations instructions:
Qt Creator app required
C++17/g++ compiler
Build steps:
1. Open the project in Qt Creator
2. Configure the kit (Desktop Qt 6.x GCC/MSVC)
3. Click Build


Execute instructions:
After building, just click the run button in the QT Creator app


Features List:
Create new diary entries
Edit and delete entries
Save entries
Load previously made entries
Track data of entries(Date made, Date modified)
Simple and clean UI design


Input / File Requirements:
Users must enter a title and content for each entry


Sample Usage / Example Run:
1. Launch the app
2. Click "New Entry"
3. Enter title and text
4. Click "Save"
5. Entry appears in the list


Known Issues / Limitations:
Changing password requires the laptop to use a password lock screen, it won't work with pin or face ID


Dependencies:
- Qt 6.x (Qt Core and Qt Widgets modules)
- C++17 compatible compiler (g++ / MSVC)
