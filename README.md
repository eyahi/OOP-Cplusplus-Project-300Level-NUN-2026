##  Overview
ZyLog v1 is a robust, GUI-based solution designed to modernize inventory tracking and sales processing for small to medium-sized enterprises. By combining an intuitive interface with powerful backend logic, ZyLog ensures seamless day-to-day operations, from stock management to financial reporting.

##  Key Features
- **Smart Inventory Control**: Real-time tracking, product categorization, and automated low-stock alerts.
- **Integrated Point of Sale**: Streamlined checkout process with dynamic cart management and discount application.
- **Actionable Analytics**: Comprehensive sales history, revenue visualization, and CSV data export.
- **Secure Architecture**: Encrypted user authentication and reliable local data persistence.
- ** High Efficiency**: Built on C++ and OpenGL for lightweight, lag-free performance.

##  System Requirements
- **OS**: Windows 10 / 11
- **Hardware**: OpenGL 3.0+ compatible graphics card
- **Compiler**: GCC/G++ (MinGW recommended for Windows) with Make support

##  Build Instructions
To compile the project from source, ensure you have `g++` and `make` installed.

1.  **Open Terminal**: Navigate to the project root directory.
2.  **Compile**: Run the following command:
    ```bash
    make
    ```
    This will compile all source files and link dependencies.
3.  **Run**: Execute the generated application:
    ```bash
    ./zylog.exe
    ```
