# ToDo Manager Application

## Description
This project is a simple ToDo Manager application developed using C++ and the Qt framework.
It allows users to add and delete tasks through a graphical user interface.

## Compilation Instructions
This project was built using Qt Creator.
To compile, open the project in Qt Creator and click the Build button.


## Execution Instructions
Run the application from Qt Creator by clicking the Run button
or execute the compiled binary:
./ToDoManager

## Features
- Add new tasks
- Delete existing tasks
- Graphical user interface using Qt
- Event-driven programming with signals and slots

## Sample Usage
1. Launch the application
2. Enter a task in the input field
3. Click the Add button to add the task
4. Select a task and click Delete to remove it

## Known Limitations
- Tasks are not saved after closing the application
- No task editing feature

## Dependencies
- Qt Framework
- C++17 compatible compiler
