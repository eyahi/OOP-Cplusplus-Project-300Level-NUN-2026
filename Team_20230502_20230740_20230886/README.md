# ToDoApp - CPE 303 Project

**ToDoApp** is a cross-platform desktop application developed as part of the **CPE 303** course. It helps users efficiently manage their daily tasks through a clean and intuitive graphical user interface built with the **Qt Framework**.

---

## Features
- **Task Management:** Add, view, and organize daily tasks.
- **Interactive UI:** User-friendly interface designed with Qt Widgets.
- **C++ Backend:** High-performance logic for task operations.
- **Cross-Platform:** Built using CMake for compatibility across different operating systems.

---

## User Guide
1. **Add a Task:** Type your task into the input field and click the **Add** button.  
2. **Complete a Task:** Select or check the task once it is finished.  
3. **Remove a Task:** Select the task you wish to delete and click the **Remove** or **Delete** button.

## Getting Started
To run this project locally, follow these steps:

1. **Requirements:** Ensure you have **Qt Creator** and a **C++ compiler** installed.  
2. **Open Project:** Open `CMakeLists.txt` in Qt Creator.  
3. **Configure:** Let CMake configure the project for your specific kit.  
4. **Build & Run:** Press the **Run** button (Green Play Icon) to build and launch the app.
